# إمكانيات النظام

يمتلك النظام مجموعة من الميزات المؤسسية (Enterprise Features) الجاهزة للاستخدام:

### 1. الإدارة الموديولية (Modularized Core)
* فصل كامل لكل وظيفة في "موديول" مستقل.
* استقلالية الأصول (Assets) وقواعد البيانات لكل موديول.
* إدارة الاعتمادات بين الموديولات لمنع تعطل النظام.

### 2. نظام إعدادات ديناميكي (System Settings)
* تخزين الإعدادات في قاعدة البيانات مع دعم Redis للسرعة الفائقة.
* تشفير تلقائي للقيم الحساسة (مثل مفاتيح الـ API).
* نمط Singleton يضمن تحميل الإعدادات مرة واحدة فقط لكل طلب (Request).

### 3. الأرشفة الذكية (Audit Logs & Activity)
* تتبع تلقائي لكل العمليات (إنشاء، تعديل، حذف).
* استخدام طوابير Redis لضمان عدم تأثر سرعة النظام.
* تمويه تلقائي للبيانات الحساسة (Masking) قبل الأرشفة.

### 4. نظام المستخدمين المتقدم
* دعم المصادقة الثنائية (2FA) عبر TOTP.
* إدارة حالات المستخدمين (نشط، معلق، محظور) مع توثيق الأسباب.
* دعم كامل للغات المتعددة والتوقيت المحلي (Timezone) والمظهر (Dark/Light).

### 5. النسخ الاحتياطي والاستعادة (Backup & Restore)
* إنشاء نسخ احتياطية لقاعدة البيانات والملفات بضغطة زر.
* استعادة النظام مع وضع الصيانة التلقائي ونسخة احتياطية احترازية قبل البدء.

### 6. الواجهة الأمامية (Modern UI)
* دعم كامل لواجهة Velzon مع نظام RTL متطور.
* تحميل ذكي للأصول (Vite Multi-entry) يشمل فقط ما يحتاجه الموديول الحالي.
