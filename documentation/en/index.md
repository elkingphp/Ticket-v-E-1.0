# System Documentation (Digilians Enterprise Framework)

Welcome to the official documentation of the Enterprise System built with Laravel 12. This system is designed to handle complex operations with high efficiency and unlimited scalability thanks to its modular architecture.

## Documentation Content
1. [System Capabilities](capabilities.md) - Overview of what the system can do.
2. [Architecture](architecture.md) - Explanation of layers and technologies used.
3. [Module Management](modules.md) - How to manage and create new modules.
4. [Permissions System](permissions.md) - Explanation of advanced roles and permissions.
5. [Audit & Logging](logging.md) - Activity tracking and security.
6. [API Guide](api.md) - How to interact with the APIs.

## Tech Stack
* **Framework**: Laravel 12
* **Database**: PostgreSQL (with JSONB support)
* **Caching/Queues**: Redis
* **Broadcasting**: Laravel Reverb
* **Frontend**: Velzon (RTL Support)
* **Architecture**: Modular Monolith (Layered Architecture)
