التثبيت والتشغيل

لضمان عمل النظام بكفاءة، يجب استخدامه ضمن البيئة التقنية الموصى بها.

1. متطلبات الخادم (Server Side)

  • PHP: الإصدار 8.2 أو أحدث.
  • Database: PostgreSQL 14+.
  • Web Server: Nginx أو Apache.
  • Memory: 4GB RAM كحد أدنى.

2. متطلبات المستخدم (Client Side)

لا يتطلب النظام تثبيت أي برامج على جهاز المستخدم، يكفي وجود متصفح إنترنت حديث:

  • Google Chrome (الموصى به).
  • Microsoft Edge.
  • Firefox.

3. خطوات تسجيل الدخول الأول

  1. افتح المتصفح وتوجه للرابط المزود من قبل الإدارة.
  2. أدخل بريدك الإلكتروني المؤسسي.
  3. أدخل كلمة المرور المؤقتة.
  4. سيطلب منك النظام تغيير كلمة المرور فوراً لأغراض أمنية.
أمن البيانات: تجنب استخدام شبكات الواي فاي العامة عند الدخول للنظام، ويفضل دائماً تفعيل خاصية التحقق بخطوتين (2FA) المتاحة في إعدادات الحساب.
