# دليل الواجهات البرمجية (API Documentation)

يدعم النظام واجهات برمجية (Stateless API) مؤمنة ومصممة للاستجابة السريعة.

## المعايير
* **Format**: JSON حصراً.
* **Authentication**: دعم Laravel Sanctum أو JWT (حسب موديول الـ Auth المختار).
* **Versioning**: يتم تنظيم الطلبات عبر `/api/v1/...`.

## توزيع الـ API
كل موديول يحتوي على ملف `Routes/api.php` مستقل. يتم تجميع هذه الملفات تلقائياً من قبل النظام.

## الهيكل الافتراضي للاستجابة
```json
{
    "success": true,
    "data": { ... },
    "message": "تمت العملية بنجاح",
    "meta": {
        "timestamp": "2026-02-12T..."
    }
}
```

## معالجة الأخطاء
يتم توحيد معالجة الأخطاء عبر النظام الأساسي لتعيد أكواد HTTP صحيحة (401, 403, 404, 422, 500) مع رسائل واضحة باللغتين العربية والإنجليزية.
