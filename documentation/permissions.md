# نظام الصلاحيات والأدوار

يعتمد النظام على حزمة `spatie/laravel-permission` مع تخصيصات معمقة لدعم المعمارية الموديولية.

## المميزات الأساسية
* **Modular Permissions**: كل صلاحية مرتبطة بموديول معين (عمود `module` في جدول الصلاحيات).
* **Granular Control**: التحكم في العمليات (عرض، إضافة، تعديل، حذف) عبر الـ Policies.
* **Role Based Access**: توزيع الصلاحيات عبر أدوار (Roles).

## الـ Policies
يتم استخدام الـ Policies داخل طبقة الـ Application لكل موديول. مثال:
* `UserPolicy`: يتحكم في إدارة أعضاء النظام.
* `SystemPolicy`: يتحكم في إعدادات النظام.

## الاستخدام البرمجي
داخل الواجهات (Blade):
```php
@can('users.create')
    <!-- زر الإضافة -->
@endcan
```

داخل المتحكمات (Controllers):
```php
$this->authorize('view', $user);
```

## التحديث التلقائي
عند تفعيل موديول جديد، ينصح بتشغيل الهجرات (Migrations) الخاصة به والتي يجب أن تحتوي على الصلاحيات المطلوبة لهذا الموديول.
