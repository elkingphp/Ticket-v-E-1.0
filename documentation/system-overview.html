نظرة عامة على النظام

يعتمد نظام ETEMS على معمارية برمجية متطورة تضمن الأداء العالي والأمان للبيانات التعليمية والإدارية.

الموديولات البرمجية

الموديول الوظيفة الرئيسية
Tickets إدارة وبناء تذاكر الدعم الفني، التواصل بين الطلاب والوكلاء.
Education إدارة المسارات، البرامج، الجداول، المجموعات، وملفات الطلاب.
Attendance تتبع حضور وغياب الطلاب في المحاضرات لحظياً.
Audit Logs سجل رقابي يسجل كل صغيرة وكبيرة لضمان النزاهة.
Roles & Permissions محرك التحكم في الوصول وتوزيع الصلاحيات.

التكنولوجيا المستخدمة

تم بناء النظام باستخدام تقنيات متطورة تشمل الـ Laravel كإطار عمل خلفي، مع قاعدة بيانات PostgreSQL لضمان استقرار البيانات الضخمة، واجهة مستخدم تعتمد على Bootstrap و CSS حديث.
