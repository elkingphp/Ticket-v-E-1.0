# توثيق النظام (Digilians Enterprise Framework)

أهلاً بكم في التوثيق الرسمي للنظام البرمجي المؤسسي المبني باستخدام Laravel 12. هذا النظام مصمم ليدير العمليات المعقدة بكفاءة عالية مع قابلية توسع غير محدودة بفضل المعمارية الموديولية.

## محتويات التوثيق
1. [إمكانيات النظام](capabilities.md) - نظرة عامة على ما يمكن للنظام فعله.
2. [البنية المعمارية](architecture.md) - شرح الطبقات (Layers) والتقنيات المستخدمة.
3. [إدارة الموديولات](modules.md) - كيف تدير وتنشئ موديولات جديدة.
4. [نظام الصلاحيات](permissions.md) - شرح للأدوار والصلاحيات المتقدمة.
5. [نظام الأرشفة والتسجيل](logging.md) - تتبع الأنشطة والأمان.
6. [دليل الـ API](api.md) - كيفية التعامل مع الواجهات البرمجية.

## التقنيات الأساسية (Tech Stack)
* **Framework**: Laravel 12
* **Database**: PostgreSQL (مع دعم JSONB)
* **Caching/Queues**: Redis
* **Broadcasting**: Laravel Reverb
* **Frontend**: Velzon (RTL Support)
* **Architecture**: Modular Monolith (Layered Architecture)
