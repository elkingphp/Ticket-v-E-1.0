بناء موديول "الأخبار" (News) بالمعمارية الصحيحة

الرئيسية إنشاء موديول الهيكل البرمجي المسارات (Routes) المتحكمات (Controllers) الصلاحيات الإعدادات مثال CRUD واجهة Velzon English

مرحلة الـ Infrastructure

1. إنشاء الـ Model:

class Article extends Model {
    use LogsActivity; // لتتبع التغييرات آلياً
    protected $fillable = ['title', 'content'];
}

2. إنشاء الـ Repository:

class ArticleRepository extends BaseRepository implements ArticleRepositoryInterface {
    // استخدم BaseRepository للحصول على find, create, update مجاناً
}

مرحلة الـ Application

3. إنشاء الـ Service:

class ArticleService extends BaseService {
    public function publishArticle(array $data) {
        // منطق النشر، إرسال إشعارات، إلخ...
        return $this->repository->create($data);
    }
}

مرحلة الـ Http

4. إنشاء الـ Controller:

public function store(ArticleRequest $request) {
    $this->service->publishArticle($request->validated());
    return $this->successResponse('تم النشر بنجاح');
}
نظام الأرشفة: لاحظ استخدام LogsActivity في الموديل؛ هذا سيجعل النظام يسجل تلقائياً "قام فلان بإضافة مقال بعنوان كذا" في الـ Audit Logs بدون أي سطر كود إضافي.
