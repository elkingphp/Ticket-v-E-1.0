نظام إعدادات النظام

إدارة متغيرات الموديول بمرونة وأداء عالٍ

الرئيسية إنشاء موديول الهيكل البرمجي المسارات (Routes) المتحكمات (Controllers) الصلاحيات الإعدادات مثال CRUD واجهة Velzon English

استخدام SettingsService

يوفر موديول Core خدمة مركزية لإدارة الإعدادات المخزنة في قاعدة البيانات والمحمية في Redis.

1. حفظ إعداد جديد

يمكنك عمل ذلك في Seeder أو Controller:

use Modules\Core\Application\Services\SettingsService;

public function __construct(SettingsService $settings) {
    $this->settings = $settings;
}

// حفظ إعداد مشفر
$this->settings->set('api_key', '12345', 'Inventory', 'string', true);

2. جلب إعداد

$val = $this->settings->get('api_key', 'default_value');

المميزات التقنية:

  • Singleton: يتم تحميل الإعدادات مرة واحدة فقط لكل Request.
  • Flash Cache: يتم تحديث الذاكرة السريعة (Redis) فور تغير أي قيمة.
  • Auto-Encryption: القيم الحساسة تشفر بـ AES-256 آلياً إذا حددت is_encrypted = true.

أنواع البيانات المدعومة:

string, integer, boolean, array, json

نصيحة: استخدم الإعدادات للقيم التي تتغير بواسطة الأدمن (مثل تفعيل ميزة معينة أو ربط API خارجي)، ولا تستخدمها لحفظ بيانات متغيرة باستمرار.
