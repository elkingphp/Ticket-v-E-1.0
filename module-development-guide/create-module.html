كيفية إنشاء موديول جديد

الخطوات الأساسية لبناء وحدة برمجية مستقلة

الرئيسية إنشاء موديول الهيكل البرمجي المسارات (Routes) المتحكمات (Controllers) الصلاحيات الإعدادات مثال CRUD واجهة Velzon English

الخطوة الأولى: استخدام أمر Artisan

نستخدم حزمة nwidart/laravel-modules لإنشاء الهيكل الأساسي. قم بتشغيل الأمر التالي في الطرفية (Terminal):

php artisan module:make Inventory

هذا الأمر سينشئ مجلد Inventory داخل مجلد Modules الرئيسي.

  • إعداد autoloading

    تأكد من تحديث ملف composer.json داخل الموديول الجديد ليعمل مع نظام الطبقات لدينا. يجب أن يكون قسم الـ psr-4 كالتالي:

    "autoload": {
    "psr-4": {
        "Modules\\Inventory\\": "./"
    }
}

  • تحديث المجلدات

    قم بإنشاء مجلدات الطبقات الأساسية كما هو متفق عليه:

    • Domain/Models, Domain/Interfaces
    • Application/Services, Application/Policies
    • Infrastructure/Repositories, Infrastructure/Database/Migrations

  • تشغيل الأمر

    بعد تغيير مسارات الـ autoload، قم بتشغيل الأمر:

    composer dump-autoload
نصيحة: دائماً ابدأ بإنشاء الهجرات (Migrations) في مجلد Infrastructure/Database/Migrations لتعريف قواعد البيانات الخاصة بموديولك.
