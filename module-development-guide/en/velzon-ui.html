Extending the Master Layout

The Core module provides a ready-to-use Master Layout with RTL support:

@extends('core::layouts.master')

@section('title', 'Product List')

@section('content')
    <div class="card">
        <div class="card-body">
            Page content here...
        </div>
    </div>
@endsection

Loading Assets

Inside your module, use the @vite directive to load your specific SASS and JS files without interfering with the system:

@push('styles')
    @vite(['Modules/Inventory/Resources/assets/sass/app.scss'])
@endpush

Ready-made Velzon Components

  • Cards: Use card and card-body classes.
  • Buttons: Use btn btn-primary.
  • Forms: Use form-label and form-control.
RTL Support: You don't need to write direction-specific CSS; the master layout automatically switches direction based on the user's selected interface language.
