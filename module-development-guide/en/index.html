Welcome to the Developer's Guide

Our system relies on an advanced modular architecture aimed at separating concerns and facilitating horizontal scalability. Instead of placing all code in one location, functions are divided into independent modules.

Important Note: Adherence to the Layered Architecture is not optional; it is the pillar that ensures system stability and ease of maintenance.

What makes our module system special?

  • Independence: Each module owns its database, interfaces, and assets.
  • Security: A centralized permissions system interacts with each module independently.
  • Performance: Integration with Redis and Queues ensuring fast response times.
  • Organization: Following Repository and Service patterns to centralize business logic.
