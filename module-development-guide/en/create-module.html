Step 1: Using the Artisan Command

We use the nwidart/laravel-modules package to create the basic structure. Run the following command in your terminal:

php artisan module:make Inventory

This command will create an Inventory folder inside the main Modules directory.

  • Set up Autoloading

    Ensure you update the composer.json file inside the new module to work with our layered system. The psr-4 section should look like this:

    "autoload": {
    "psr-4": {
        "Modules\\Inventory\\": "./"
    }
}

  • Update Directories

    Create the core layer directories as agreed:

    • Domain/Models, Domain/Interfaces
    • Application/Services, Application/Policies
    • Infrastructure/Repositories, Infrastructure/Database/Migrations

  • Run the Command

    After changing autoload paths, run the following command:

    composer dump-autoload
Tip: Always start by creating migrations in the Infrastructure/Database/Migrations folder to define your module's database schema.
