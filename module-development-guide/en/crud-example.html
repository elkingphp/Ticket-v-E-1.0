Infrastructure Layer

1. Create Model:

class Article extends Model {
    use LogsActivity; // For automatic change tracking
    protected $fillable = ['title', 'content'];
}

2. Create Repository:

class ArticleRepository extends BaseRepository implements ArticleRepositoryInterface {
    // Use BaseRepository to get find, create, update for free
}

Application Layer

3. Create Service:

class ArticleService extends BaseService {
    public function publishArticle(array $data) {
        // Publishing logic, sending notifications, etc...
        return $this->repository->create($data);
    }
}

Http Layer

4. Create Controller:

public function store(ArticleRequest $request) {
    $this->service->publishArticle($request->validated());
    return $this->successResponse('Published Successfully');
}
Audit System: Note the use of LogsActivity in the Model; this will automatically log "User X added article titled Y" in the Audit Logs without any extra line of code.
