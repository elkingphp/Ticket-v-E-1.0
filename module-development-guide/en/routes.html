Where to find route files?

Routes are located in the Routes folder inside each module, divided into:

  • web.php: For UI routes (includes Sessions and CSRF).
  • api.php: For API routes (includes JSON and Stateless auth).

Example of adding a Web Route:

use Illuminate\Support\Facades\Route;
use Modules\Inventory\Http\Controllers\ItemController;

Route::prefix('inventory')->group(function() {
    Route::get('/', [ItemController::class, 'index'])->name('inventory.index');
    Route::post('/store', [ItemController::class, 'store'])->name('inventory.store');
});

Applying Permission Middleware:

To protect a route and enable it only for users with specific permissions:

Route::middleware(['auth', 'can:inventory.view'])->group(function() {
    Route::get('/inventory', [ItemController::class, 'index']);
});

Protecting the Entire Module

Our system provides a special middleware to check if the module is enabled before access:

Route::middleware(['module_status:Inventory'])->group(function() {
    // All routes here will stop if the Inventory module is disabled
});
Warning: Always use Named Routes to facilitate linking in views, e.g., inventory.list.
