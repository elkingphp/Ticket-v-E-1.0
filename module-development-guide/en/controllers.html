Where to place Controllers?

Controllers should be placed in the Http/Controllers folder inside the module.

Golden Rule

A Controller should not contain complex logic or direct DB queries. It should call the "Service".

namespace Modules\Inventory\Http\Controllers;

use Modules\Core\Http\Controllers\BaseController;
use Modules\Inventory\Application\Services\ItemService;
use Modules\Inventory\Http\Requests\SaveItemRequest;

class ItemController extends BaseController
{
    protected $itemService;

    public function __construct(ItemService $service)
    {
        $this->itemService = $service;
    }

    public function store(SaveItemRequest $request) 
    {
        // 1. Check Permissions (optional here, preferred in Policy)
        $this->authorize('create', Item::class);
        
        // 2. Process Data via Service
        $item = $this->itemService->createItem($request->validated());

        // 3. Return Response
        return redirect()->back()->with('success', 'Item Added Successfully');
    }
}

Using Form Requests

Always accept data via a dedicated FormRequest located in Http/Requests to ensure input security before reaching the controller.

Note: Always inherit from BaseController found in the Core module to utilize unified response features.
