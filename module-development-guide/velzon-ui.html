تطوير الواجهات بـ Velzon

استخدام القالب الموحد مع دعم RTL

الرئيسية إنشاء موديول الهيكل البرمجي المسارات (Routes) المتحكمات (Controllers) الصلاحيات الإعدادات مثال CRUD واجهة Velzon English

وراثة القالب الرئيسي (Master Layout)

موديول Core يوفر القالب الرئيسي الجاهز بـ RTL:

@extends('core::layouts.master')

@section('title', 'قائمة المنتجات')

@section('content')
    <div class="card">
        <div class="card-body">
            محتوى الصفحة هنا...
        </div>
    </div>
@endsection

تحميل الأصول (Assets)

داخل الموديول الخاص بك، استخدم توجيه @vite لتحميل ملفات الـ SASS والـ JS الخاصة بك بدون التداخل مع النظام:

@push('styles')
    @vite(['Modules/Inventory/Resources/assets/sass/app.scss'])
@endpush

مكونات Velzon الجاهزة

  • البطاقات (Cards): استخدم كلاس card و card-body.
  • الأزرار: استخدم btn btn-primary.
  • النماذج: استخدم form-label و form-control.
دعم RTL: لا تحتاج لكتابة كود CSS خاص للاتجاه؛ القالب الرئيسي يقوم بتبديل الاتجاه تلقائياً بناءً على لغة واجهة المستخدم المحددة في النظام.
